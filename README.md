Address Book Management System (C)

Description:
A console-based application developed in C that allows users to manage contacts
using file handling and structured programming.

Features:
- Add new contacts
- Search contacts by name or number
- Update existing contact details
- Delete contacts
- Persistent storage using files

Technologies Used:
- C Programming
- Structures
- File Handling
- Modular Programming

How to Compile and Run:
gcc *.c
./a.out

Learning Outcomes:
- Hands-on experience with structures and pointers
- File I/O operations in C
- Modular code design
